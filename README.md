# Scoop Bucket

A bucket for [Scoop](https://scoop.sh), the Windows command-line installer.

## 特色

* 包含中国人常用的软件
* 修改来自官方 bucket 的一些软件，使其更加便携化
* 不需要管理员权限
* 尽量少的修改系统环境变量
* 尽量少的创建 shim

## 用法

1. 添加此 Bucket：`scoop bucket add wzv5 https://github.com/wzv5/ScoopBucket`
2. 安装软件：`scoop install wzv5/<app_name>`
